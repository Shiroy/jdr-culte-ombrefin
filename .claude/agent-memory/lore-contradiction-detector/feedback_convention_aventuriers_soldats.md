Dans cet univers, "aventuriers" et "soldats mandatés par le Comte" sont des termes interchangeables pour désigner les joueurs (PJ). C'est une convention narrative délibérée, pas une incohérence de lore.

**Why:** L'auteur a confirmé explicitement que ces deux formulations coexistent par choix. Les signaler comme ambiguïté crée du bruit inutile dans les rapports d'audit.

**How to apply:** Lors de tout audit futur, ignorer toute divergence entre ces deux formulations. Ne pas les mentionner, même comme ambiguïté non bloquante.
