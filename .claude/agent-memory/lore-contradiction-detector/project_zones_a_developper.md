## Données de référence d'échelle introduites par PONT_SUR_DOUVE.md (2026-03-22)

Ces valeurs sont les premières données chiffrées de l'univers — les utiliser comme référence lors des futures fiches.
- Population d'un bourg capital de sous-région : ~10 000 habitants
- Garnison d'un comte de sous-région : 150-200 soldats professionnels

**Why:** Identifié lors de l'audit du 2026-03-22. Les zones peu détaillées sont les premières sources de contradictions lors des ajouts de contenu.
